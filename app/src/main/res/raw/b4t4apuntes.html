Tema 4: Administración de redes de área local

Administración de redes

La administración de redes se basa en la gestión y monitorización de los dispositivos y recursos de una red para asegurar su correcto funcionamiento.

SNMP (Simple Network Management Protocol)

SNMP es un protocolo estándar de internet utilizado para gestionar y supervisar dispositivos de red como routers, switches, servidores e impresoras.

Permite recolectar información sobre rendimiento, diagnosticar problemas y controlar dispositivos.

Puertos: 161/UDP (agente) y 162/UDP (gestor).

Herramientas de gestión de redes

  • Nagios: Monitorización de redes y servidores.
  • Zabbix: Plataforma avanzada de monitorización.
  • SolarWinds: Solución integral para gestión de redes.

SNMPv1

Primera versión de SNMP, definida en el RFC 1157.

Los RFC (Request for Comments) definen los estándares y especificaciones de SNMP.

  • Basado en una estructura simple de comunidad para autenticación.
  • Usa comandos GET, SET y TRAP.
  • No proporciona encriptación ni autenticación segura.

SNMPv2

SNMPv2 introdujo mejoras sobre SNMPv1, pero inicialmente su modelo de seguridad era complejo.

Se definió en los RFC 1441-1450 y posteriormente nació SNMPv2c, que es la versión más utilizada.

  • Soporte para nuevos tipos de datos.
  • Mejor eficiencia en la recuperación de datos.
  • SNMPv2c mantiene el modelo de comunidad de SNMPv1 pero mejora la gestión.

SNMPv3

Incluye características avanzadas de seguridad, como autenticación y encriptación.

Su configuración es más compleja debido a estos aspectos de seguridad.

LDAP

LDAP (Lightweight Directory Access Protocol) es un protocolo de comunicación utilizado para acceder y gestionar servicios de directorios, como bases de datos que contienen información sobre usuarios, dispositivos o recursos en una red. Se usa comúnmente en sistemas para gestionar usuarios y autenticar accesos, por ejemplo, en servicios como Active Directory de Microsoft o en sistemas de correo electrónico.

Es ideal para almacenar información de usuarios, grupos y permisos.

  • Versión: 3
  • Protocolo de transporte: TCP/IP

Active Directory

Basado en LDAP, proporciona servicios de directorio en entornos Windows.

Seguridad en LDAP

  • LDAPS: Implementación de LDAP sobre SSL/TLS para comunicación segura.
  • Autenticación y Autorización: LDAP gestiona la validación y los permisos de acceso.
  • ACL (Access Control Lists): Controla quién puede leer o modificar el directorio.
